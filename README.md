📊 Pneumonia Detection using Convolutional Neural Networks (CNN)
📌 Descripción
Este proyecto desarrolla la técnica de Redes Neuronales Convolucionales aplicado imagenes de torax incluyendo normalización y aumento de datos (data augmentation) para mejorar la capacidad de generalización del modelo.

Fuente: Dataset de imagenes de Rayos X adjunto en el repositorio

⚙️ Objetivo
El objetivo es detectar pacientes con neumonía mediante imágenes de rayos X de su tórax.
